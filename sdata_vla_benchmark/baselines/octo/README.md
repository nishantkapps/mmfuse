# Octo (frozen)

Use the public Octo model weights and policy API. Map native actions or language outputs to SData classes `0..7` **after** inference (post-hoc; no new head on Octo).

See the [Octo project](https://github.com/octo-models/octo) for installation.

Output JSON must match the schema in `baselines/openvla/README.md` (`model` field: `octo`).
